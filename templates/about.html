About & Donate

smossbot is a Twitch bot self-developed by me, Omar (moz).

It started as a passion project that was actually intended to run on your local machine as a downloadable program, but was later adopted under the vision of providing the functionality from a website, with no downloads required, and virtually no resources needing to be used from your PC.

The website's frontend and backend, and the Twitch bot that appears in your chat from this website, is all self-developed, apart from the frameworks/libraries used of course. The website is built on Django, and the Twitch bot itself is built using the TwitchIO Python library.

This project quickly became one of (and my main) project that I worked on, and I have had a lot of fun learning everything and picking up a lot of technologies that were foreign to me. I built smossbot because my friends and I wanted functionalities that weren't readily accessible or were really inconvenient to get working; smossbot quickly expanded to provide a bunch of other niche little functionalities.

I've never made a big project like this, but it's helped me learn more about web development and grow, personally. Because of this being a passion project, I always wanted this to be a completely free resource with no crazy paywalls, so that's what it'll remain as.
