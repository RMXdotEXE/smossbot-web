smossbot's commands for the MODS
!ping
smossbot responds with  Pong!  to let you know it's active and listening.
Permission level: Viewer
!songskip
Manually skips the song currently playing.
Permission level: Moderator+
!variable <variablename> <value>
Updates various values that tweak how certain parts of smossbot work, so you can fine-tune behaviours.
variablename value Purpose
songreq:emote Any The emote that appears before the song request message.
Default: Okayge
chatgpt:clearsize [1,2,3,...] An integer (whole number) representing how many past exchanges should be cleared at a time until ChatGPT can continue on with more conversations.
Default: 2
chatgpt:showfull True OR False If set to "true", smossbot will not cut off the response from ChatGPT, which will lead to a lot of messages from smossbot. Precisely, smossbot will send as much Twitch messages as it needs to post the complete ChatGPT response, typically a maximum of 5. If set to "false" (or anything else), it'll cut it off and replace the end with "... (max 500)".
Default: false
chatgpt:maxprompttokens [1,2,3,...] An integer (whole number) representing how many tokens ChatGPT can use for the chat completion. A shorter number means the responses are forced to be more concise, and a larger number means the responses are more thorough. If you set the number too short for responses to be shorter though, ChatGPT may give you a garbage/junk response. For reference, 256 is the minimum recommended amount of tokens ChatGPT needs to supply a successful response.
Default: 256
gptimage:imagesize [1,2,3] An integer (whole number) between 1 and 3, inclusive, representing the size of generated images by ChatGPT. The sizes, respectively, are: 256x256, 512x512, and 1024x1024.
Default: 2
songskip:emote Any The emote that appears before the song skip message.
Default: PepeLaugh
ytreq:maxsecs [1,2,3,...] An integer (whole number) representing how many seconds, at the maximum, incoming YouTube requests should play for.
Default: 60
ytreq:mutespotify "true" OR "false" If set to "true", incoming YouTube requests will pause your Spotify playback for the duration of the videos. If multiple videos are queued, Spotify will only resume once all queued videos are done.
Default: true
Permission level: Moderator+ OR xzmozxx
