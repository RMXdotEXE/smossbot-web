!ping
smossbot responds with Pong! to let you know it's active and listening.

Permission level: Viewer

!variable <variablename> <value>
Updates various values that tweak how certain parts of smossbot works in your chat.
variablename value Purpose
songreq_emote Any The emote that appears before the song request message.
Default: Okayge
chatgpt_clearsize [1,2,3,...10] An integer (whole number) representing how many past exchanges should be cleared at a time until ChatGPT
can continue on with more conversations.
Default: 2
chatgpt_full "true" OR Any If set to "true", smossbot will not cut off the response from ChatGPT, which will lead to a lot of
messages from smossbot. Precisely, smossbot will send as much Twitch messages as it needs to post the complete
ChatGPT response, typically a maximum of 5. If set to "false" (or anything else), it'll cut it off and replace
the end with "... (max 500)".
Default: true
